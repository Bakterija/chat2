# chat2
##API     

### Set nick     
NICK 'username'  
### Join chan     
JOIN '#chan'
### Leave chan
LEAVE '#chan'
### Leave server
QUIT					
### Get usernames from chan      
NAMES '#chan'					
### Send msg    
MSG target('usrname' or #chan) :'text'
### Get chan or user info
WHO target('usrname' or #chan)
